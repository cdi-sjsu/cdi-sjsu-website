# CDI Image Placeholders

Drop real club images into this folder and update the matching placeholder blocks in the HTML pages.

When deploying with the GitHub Pages workflow, Vite copies this folder to `dist/images`.
Use relative image paths in HTML, such as `images/project-vga-controller.jpg`, so project-page URLs like
`https://username.github.io/repo-name/` work correctly.

Suggested filenames:

- `cdi-logo-original.png`
- `cdi-logo-transparent.png`
- `project-vga-controller.jpg`
- `project-shift-comparator.jpg`
- `gallery-build-night-01.jpg`
- `gallery-pcb-layout-01.jpg`
- `gallery-tinytapeout-review-01.jpg`

Use landscape images around 1600x1000 for project and gallery areas when possible.
